# Gerenciador-Apostas-Esportivas
Apresentamos o Gerenciador de Apostas Esportivas: um software completo para o gerenciamento, análise e prospecção de dados em SureBets para exchanges esportivas. Desenvolvido como um projeto pessoal, esse aplicativo Python foi projetado com o objetivo de fornecer um controle eficiente e uma análise aprofundada de dados, além de oferecer recursos gráficos e gerenciamento de SureBets.

<img width="400" alt="Captura de Tela 2023-06-20 às 03 31 00" src="https://github.com/seblutzer/Gerenciador-Apostas-Esportivas/assets/114627479/b522c5e8-5765-40ff-a143-651bb78fe291">

Com uma interface intuitiva, o software permite a adição fácil e cálculo automático de mais de 15 tipos de SureBets além do controle preciso das BetHouses.

<img width="400" alt="Captura de Tela 2023-06-20 às 03 33 08" src="https://github.com/seblutzer/Gerenciador-Apostas-Esportivas/assets/114627479/ab25ad46-2b27-49ef-9409-556c6b6fe9de">
<img width="534" alt="Captura de Tela 2023-06-20 às 03 33 29" src="https://github.com/seblutzer/Gerenciador-Apostas-Esportivas/assets/114627479/fd59cc03-b5e1-4ae9-913c-b71182045571">


Uma funcionalidade essencial é o treeview, que exibe de forma clara todas as SureBets já realizadas e armazenar os resultados. Além disso, foram implementados filtros personalizáveis para visualização das SureBets feitas, proporcionando uma análise mais precisa e relevante.

<img width="400" alt="Captura de Tela 2023-06-21 às 03 41 12" src="https://github.com/seblutzer/Gerenciador-Apostas-Esportivas/assets/114627479/2d3bbeaf-ee28-4080-bfc5-5cf1f5bfcadc">

O Gerenciador de Apostas Esportivas também oferece controle e estatísticas detalhadas de cada BetHouse, permitindo que você tome decisões informadas e estratégicas. A integração com o banco de dados SQL garante um armazenamento eficiente e seguro dos dados, facilitando a recuperação e manipulação dos mesmos.

<img width="400" alt="Captura de Tela 2023-06-20 às 03 32 18" src="https://github.com/seblutzer/Gerenciador-Apostas-Esportivas/assets/114627479/a1c29e43-f678-4c06-a13a-8484ac69fbfc">

Com o objetivo de fornecer uma visualização abrangente dos dados, foram criados oito gráficos interativos, que podem ser filtrados por data. Essa funcionalidade permite uma análise visual mais aprofundada, identificando tendências e padrões importantes.
